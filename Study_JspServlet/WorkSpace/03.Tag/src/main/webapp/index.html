JSP

여기서 배우는 내용은 개념이나 해당하는게 있다 존재여부가 중요하지 외우거나 할필요없음

스크립트태그 ( Script Tag )

  • 태그 처음 사용(JSP처음)
  • 변수 차이 알아보기
  • 제어문(IF문)
  • 제어문(For문)
  • 제어문(For문 중첩)
  • 메소드(연습해보기 혼자공부할때) plus ,mul , minus , div

디렉티브 태그 ( Directive Tag )

  • Import 구문
  • 에러페이지 처리
  • 인클루드(붙이기)

액션 태그 ( Action Tag )

  • 액션 태그 포워드(Forward)개념이 중요함
  • Include ( 페이지 붙이기 )
